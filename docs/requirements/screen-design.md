## 2. 사이드바 구조

### 2.1 사이드바 섹션 (4개 섹션, 12개 항목)

Readout의 25개 항목 중 **zm-agent-manager에 필요한 기능만 선별**하여 12개로 구성.

```
Monitor
  ├── 📊 Dashboard        F1   대시보드 (홈)
  ├── 🔴 Live             F2   실시간 세션
  ├── 📋 Sessions         F3   세션 목록 + 타임라인
  └── 🔧 Tools            F4   도구 사용 분석

Tasks
  ├── 📌 Board            F11  태스크 보드 (칸반)
  └── 🔄 Workflow         F14  태스크 워크플로우 (Phase 3)

History
  ├── ▶️ Replay            F5   세션 리플레이 (Phase 2)
  ├── 📝 Diffs            F6   파일 변경 추적 (Phase 2)
  └── 🔍 Search           F9   전문 검색 (Phase 3)

Manage
  ├── 📄 Docs             F12/F15  문서 인벤토리 + 중요도
  ├── 💰 Costs            F13  비용 추적 (Phase 2)
  └── ⚙️ Settings         앱 설정
```

### 2.2 사이드바 동작

| 동작 | 설명 |
|------|------|
| 항목 클릭 | 콘텐츠 영역 전환 (애니메이션 없음) |
| 선택 표시 | 배경 하이라이트 (rounded rect, bg-sidebar-selected) |
| 활성 표시 | Live 항목에 빨간 점 (활성 세션 있을 때) |
| 알림 뱃지 | Docs 항목에 숫자 뱃지 (미리뷰 문서 변경 수) |
| Phase 표시 | Phase 2/3 항목은 "(곧 출시)" 텍스트로 비활성 |
| 섹션 헤더 | 회색 대문자, 12px, 클릭 불가 |

### 2.3 Phase별 사이드바 활성화

| Phase | 활성 항목 |
|-------|----------|
| Phase 1 | Dashboard, Live, Sessions, Tools, Board, Settings |
| Phase 2 | + Replay, Diffs, Docs, Costs |
| Phase 3 | + Workflow, Search |

## 4. 네비게이션 흐름

### 4.1 주요 이동 경로

```
Dashboard ──→ Live (활성 세션 클릭)
    │         │
    │         └──→ Sessions/타임라인 (세션 카드 클릭)
    │
    ├──→ Sessions ──→ 세션 타임라인 (세션 항목 클릭)
    │         │
    │         └──→ Replay (Replay 버튼 클릭)
    │
    ├──→ Board ──→ TaskDetail (태스크 카드 클릭)
    │         │
    │         └──→ Sessions/타임라인 (소속 세션 링크 클릭)
    │
    ├──→ Docs ──→ DocDiff (문서 항목 클릭)
    │
    └──→ Alerts ──→ Docs/Review (문서 알림 클릭)
                ──→ Costs/Budget (비용 알림 클릭)
```

### 4.2 뒤로가기

모든 상세 뷰(타임라인, TaskDetail, DocDiff, Replay)의 좌측 상단에 `[← Back]` 버튼.
브라우저 히스토리 패턴으로 관리 (Electron에서 hash routing).

### 4.3 딥링크

각 화면은 URL hash로 접근 가능:
```
#/dashboard
#/live
#/sessions
#/sessions/{sessionId}          → 세션 타임라인
#/tools
#/board
#/board/{taskId}                → 태스크 상세
#/replay/{sessionId}            → 세션 리플레이
#/diffs
#/docs
#/docs/{filePath}               → 문서 diff
#/costs
#/search
#/settings
```
